=== 안건 1: 프로덕트 기획 현황 ===

이수진: 플러스멤버십 기획은 90% 확정 상태입니다. 핵심 기능 세 가지입니다.

둘째, AI 맞춤 추천. 사용자 행동 데이터 기반 개인화 콘텐츠 추천. 데이터팀 협업 중.
셋째, 전용 고객센터. 전담 상담원 배정, 평균 응답 시간 목표 3분 이내.

가격 정책은 월 9,900원, 연간 결제 시 99,000원(월 8,250원 효과)으로 확정. 초기 3개월 프로모션 가격을 월 6,900원으로 할지 월 4,900원으로 할지 미정.
기업용 B2B 요금제는 v1에서 제외, 10월 이후 별도 기획 예정.

=== 안건 2: 개발 진행 상황 ===

박정우: 백엔드 전체 진행률 약 65%.
- 결제 시스템: PG사 토스페이먼츠 계약 완료, API 연동 40% 진행, 7월 15일 완료 목표.
- 구독 관리 시스템(가입/해지/갱신/환불): 70% 완료. 환불 정책(일할 계산 vs 월 단위) 법무 확인 필요.
- 콘텐츠 접근 권한: 80% 완료. 동시 접속 기기 제한(최대 3대) 구현 중.

리스크: 토스페이먼츠 정기결제 API 변경으로 일부 재작업 필요. 최악의 경우 2주 지연.

송예린: 2주 밀리면 QA 기간이 2주밖에 안 남음. 결제 테스트 시나리오만 47개. 최소 3주 필요.

박정우: 대안 두 가지.
방법 1: 외주 개발자 2명 투입, 비용 약 1,200만원 추가.
방법 2: 런칭 시 신용카드만 지원, 간편결제(카카오페이/네이버페이)는 9월 말 업데이트로 분리.

정민호: 기존 결제 데이터 기준 신용카드 58%, 카카오페이 24%, 네이버페이 15%, 기타 3%. 간편결제 제외 시 잠재 결제 약 39% 손실.

김도현: 방법 1(외주 투입)로 결정. 외주 비용은 본부장 승인 후 내일까지 결과 공유.

최유나: 프론트엔드 진행률 55%. 멤버십 가입 플로우 UI 완성, 마이페이지 구독 관리 화면 작업 중. 디자인 시안 확정 대기.

한서윤: 구독 관리 화면 디자인 6월 20일 전달 예정. 멤버십 배지 종류 확정 필요. 기존안 3종(3개월/6개월/1년) vs 이수진 제안 5종(1개월/3개월/6개월/1년/2년 이상).

김도현: 배지 5종 확정. 디자인 일정 6월 23일로 조정 승인.

정민호: 시뮬레이션 결과.
- 4,900원 프로모션 3개월: 가입자 예상 15,000명, 매출 약 2억 2,050만원, 3개월 후 정상가 전환율 예상 60%.

오지훈: 장기적으로 보면 4,900원이 가입자 풀을 키우는 데 유리합니다. 15,000명의 60%면 9,000명 유지, 10,000명의 73%면 7,300명 유지. 정상가 기준 월 매출이 8,910만원 vs 7,227만원으로 4,900원이 더 높습니다.

김도현: 설득력 있네요. 프로모션 가격 월 4,900원으로 확정합니다.

## Part 6. 최종 결론

> **"회의록 요약" 과업은 긴 문맥 유지와 형식 준수가 가장 중요하다.**
> 회의록은 다수의 안건·담당자·수치·기한이 혼재하는 복잡한 문서이므로,
> 후반부 정보까지 균등하게 유지하고 지정된 템플릿을 충실히 따르는 능력이 핵심이다.

### 🥇 채택 모델: **ChatGPT (GPT-5.3)**

```
채택 근거:
✅ 문맥 유지 (18/20): 7개 안건 전체를 균등하게 커버
✅ 형식 준수 (19/20): 지정 템플릿 6개 섹션 완벽 반영
✅ 정확성 (18/20): 기재한 수치 전부 정확, 환각 없음
✅ 경영진 보고 즉시 활용 가능한 완성도

단점 감수:
⚠️ 비용이 세 모델 중 가장 높음
⚠️ 응답 속도가 상대적으로 느림
→ 그러나 회의록 요약은 빈도가 낮고 정확성이 중요한 과업이므로
   비용·속도 단점은 감수할 수 있음
```

### 🥈 보조 활용: **Gemini 2.0 Flash**

```
활용 시나리오:
- 회의 직후 "5분 요약 브리핑"이 필요한 경우
- 비용 절감이 중요한 대량 처리 환경
- Google Meet/Docs 자동화 파이프라인 구축 시
```

### ❌ 비추천: **Perplexity Sonar**

```
비추천 이유:

  → Perplexity의 핵심 강점인 "실시간 웹 검색"이 전혀 활용되지 않음

- 단, 외부 정보 검색이 필요한 과업(시장 조사, 경쟁사 분석 등)에서는 
  강력한 도구가 될 수 있음
```
