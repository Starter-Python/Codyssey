## 안건 2: 개발 진행 상황

**박정우**

- 백엔드 전체 진행률 약 **65%**

### 결제 시스템
- PG사 토스페이먼츠 계약 완료
- API 연동 40% 진행
- 7월 15일 완료 목표

### 구독 관리 시스템
- 가입/해지/갱신/환불
- 70% 완료
- 환불 정책(일할 계산 vs 월 단위) 법무 확인 필요

### 콘텐츠 접근 권한
- 80% 완료
- 동시 접속 기기 제한(최대 3대) 구현 중

### 리스크
- 토스페이먼츠 정기결제 API 변경으로 일부 재작업 필요
- 최악의 경우 2주 지연

**송예린**
> 2주 밀리면 QA 기간이 2주밖에 안 남음. 결제 테스트 시나리오만 47개. 최소 3주 필요.

**박정우**

대안 두 가지

1. 외주 개발자 2명 투입
   - 비용 약 1,200만원 추가
2. 런칭 시 신용카드만 지원
   - 간편결제(카카오페이/네이버페이)는 9월 말 업데이트로 분리

**정민호**

기존 결제 데이터 기준

- 신용카드 58%
- 카카오페이 24%
- 네이버페이 15%
- 기타 3%

간편결제 제외 시 잠재 결제 약 39% 손실

**김도현**
> 방법 1(외주 투입)로 결정. 외주 비용은 본부장 승인 후 내일까지 결과 공유.

**최유나**
- 프론트엔드 진행률 55%
- 멤버십 가입 플로우 UI 완성
- 마이페이지 구독 관리 화면 작업 중
- 디자인 시안 확정 대기

**한서윤**
- 구독 관리 화면 디자인 6월 20일 전달 예정
- 멤버십 배지 종류 확정 필요
  - 기존안 3종(3개월/6개월/1년)
  - 이수진 제안 5종(1개월/3개월/6개월/1년/2년 이상)

**김도현**

## 안건 4: 마케팅 전략

**오지훈**

### 마케팅 3단계 계획

1. 사전 알림 (8/1~15)
   - 이메일
   - 푸시 티저 캠페인

2. 사전 예약 (8/16~31)
   - 얼리버드 할인
   - 사전 예약자 첫 달 50% 할인

3. 정식 런칭 (9/1~)
   - SNS 광고
   - 유튜브 인플루언서
   - 보도자료

### 예산

| 항목 | 예산 |
|------|------:|
| SNS 광고 | 3,000만원 |
| 인플루언서 | 2,500만원 |
| PR | 1,000만원 |
| 이벤트/프로모션 | 2,000만원 |
| **합계** | **8,500만원** |

### 프로모션 가격

- 오지훈: 월 4,900원 추천
  - 경쟁사 "프리미엄플러스" 월 7,900원 대비 경쟁력
- 이수진: 월 6,900원 주장
  - 수익성 우려

**정민호**

시뮬레이션 결과

| 구분 | 4,900원 | 6,900원 |
|------|---------:|---------:|
| 예상 가입자 | 15,000명 | 10,000명 |
| 예상 매출 | 약 2억 2,050만원 | 약 2억 700만원 |
| 정상가 전환율 | 60% | 73% |

**오지훈**
> 장기적으로 보면 4,900원이 가입자 풀을 키우는 데 유리합니다. 15,000명의 60%면 9,000명 유지, 10,000명의 73%면 7,300명 유지. 정상가 기준 월 매출이 8,910만원 vs 7,227만원으로 4,900원이 더 높습니다.

**김도현**
> 설득력 있네요. 프로모션 가격 월 4,900원으로 확정합니다.

## 안건 6: CS 운영 준비

**강하은**

전용 고객센터 운영 계획

- 전담 상담원 5명 신규 채용 예정
- 8월 초 채용 완료
- 2주 교육 후 8월 중순 투입

응답 시간 목표 3분 이내를 위해

- 챗봇 도입 검토 중
- 단순 문의(비밀번호 변경, 결제 확인 등)는 챗봇 처리
- 복잡한 문의만 상담원 연결

예상 문의량

- 런칭 첫 달 일 평균 300건
- 이 중 60%는 챗봇 처리 가능 예상

우려 사항

- 환불 관련 문의가 많을 것으로 예상
- 환불 정책이 확정되지 않아 CS 매뉴얼 작성이 지연되고 있습니다.

**김도현**
> 환불 정책은 윤재혁 님이 법적 검토 후 6월 말까지 확정해주세요.

**윤재혁**

- 전자상거래법상 구독 서비스는 중도 해지 시 일할 계산 환불이 원칙입니다.
- 다만 프로모션 기간 가입자의 경우 정상가 기준으로 이용 일수를 계산할지, 프로모션가 기준으로 할지 정해야 합니다.
- 법적으로는 둘 다 가능하지만, 프로모션가 기준이 소비자 친화적입니다.
- 6월 27일까지 최종 의견서 전달하겠습니다.
